# StudyLeave 🎓✨

> **The Ultimate AI-Powered Study Companion**

StudyLeave is a cutting-edge study management application that combines advanced AI tutoring, intelligent note-taking, and comprehensive study tools into one seamless experience. Powered by GPT-4o, it's like having a personal tutor, note organizer, and study assistant all in one.

## 🤖 **Advanced AI Features**

### **Intelligent AI Tutor**
- **GPT-4o Integration**: Powered by OpenAI's most advanced language model
- **Real-Time Streaming**: Experience natural conversations with live typing effects
- **Context-Aware Assistance**: AI understands your current subject and notes
- **Multi-Modal Chat**: Global chat input and in-page note-specific conversations
- **Smart Content Generation**: Create comprehensive study notes on any topic instantly

### **Revolutionary Text Explanation**
- **Select-to-Explain**: Highlight any text in your notes for instant AI explanations
- **Structured Learning**: Get explanations with:

  - 📚 **Detailed Explanation** - Step-by-step breakdown

## 📦 **Quick Setup**

### **Development**
```bash
# Clone and navigate to the project
cd StudyLeave

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to experience StudyLeave!

### **Environment Configuration**
Create a `.env.local` file in the root directory:

```env
OPENAI_API_KEY=your_openai_api_key_here
MODEL_NAME=gpt-4o
```

> **🔒 Security Note**: Never commit API keys to version control. The `.env.local` file is automatically ignored by Git.

## 🌐 **Deployment Options**
